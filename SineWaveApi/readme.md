Для генерации графика была использована библиотека ScottPlot.
Для получения изображения с графиком необходимо послать Post запрос с следующими параметрами(from body)
	a - амплитуда в условных единицах
	fd - частота дискретизации сигнала в точках за секунду
	fs - частота сигнала в герцах
	n - количество периодов
	height - высота изображения
	width - ширина изображения

Логика по генерации графика вынесена в отдельный сервис(SinusoidService), который встроен в контроллер через DI
Для параметров запроса настроены ограничения модели через аттрибуты Range
Для тестирования API используется Swagger
